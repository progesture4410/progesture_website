Features

Explore the powerful features of ProGesture

User Authentication

Ensures secure and controlled access to the system through verified user accounts using a QR code-based login system. Users present a unique QR code to the system camera for verification, enabling fast and seamless authentication while minimizing unauthorized access.

Once authenticated, users gain access to a personalized interface where they can upload, manage, and retrieve presentation files through the web-based platform. The system securely stores user data and enforces access control to ensure that only authorized individuals can access and present their content.

Gesture Control

Enables users to control presentations through intuitive hand movements using vision-based gesture recognition technology. The system utilizes a camera and computer vision frameworks to detect and interpret hand gestures in real time, eliminating the need for physical input devices.

Predefined gestures are mapped to specific commands such as slide navigation, media control, and zoom functions for seamless interaction. This allows instructors to move freely within the classroom while maintaining full control of their presentation.

Integrated Operation

Combines hardware and software components into a unified system that operates seamlessly without the need for external devices. The projector integrates a mini-PC, camera module, gesture recognition software, and web-based interface to handle all presentation functions internally.

This allows users to upload, access, and control presentation files directly within the system. By eliminating reliance on laptops, cables, and additional peripherals, it simplifies setup and reduces technical interruptions.
